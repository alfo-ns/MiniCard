# MiniCard - Web-Based Link & Bookmark Manager

**MiniCard** is a modern, sleek, and highly functional web application designed to organize your digital life. It allows you to save, categorize, and access your favorite links and bookmarks with unmatched ease and style. Built with Vite and Vanilla TypeScript, MiniCard combines a beautiful glassmorphism UI with powerful features like real-time search, tag filtering, and automatic favicon fetching.

## ✨ Features

-   **Stunning Glassmorphism UI**: A modern, eye-catching interface with blur effects, gradients, and smooth animations.
-   **Real-Time Search**: Find any link instantly as you type.
-   **Tag Filtering**: Organize your collection with custom tags and filter them effortlessly.
-   **Auto-Fetch Favicons**: Automatic extraction of website favicons for a professional look.
-   **Custom Images**: Upload or link your own custom images for cards.
-   **Drag-and-Drop Reordering**: Intuitive interface to organize your links.
-   **Data Persistence**: Your data is securely stored in your browser's LocalStorage.
-   **Import/Export**: Easily back up or migrate your link collection.
-   **Responsive Design**: Seamless experience across different devices.

## 🚀 Getting Started

### Prerequisites
Ensure you have **Node.js** installed (v18 or higher recommended).

### Installation
1.  Clone the repository:
    ```bash
    git clone <repository-url>
    cd MiniCard
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

### Usage
Start the development server:
```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view the application in your browser.
